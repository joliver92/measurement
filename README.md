# Measurement Package

# Installation Instructions

```
git clone https://github.com/joliver92/measurement/
pip install -e . 
```
To import into your program clone to the base directory. 
```
from measurement import Measurement 
```
# Package use 
```
x = Measurement(8,4) 
y = Measurement(2,3)
x+y = 10 +/- 5 
```
